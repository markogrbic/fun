The gist code is contained in a nut.js file which has tho functions - foo() and bar(). The foo function is implemented and it takes as input text and encodes it in base64 and appends some data to it. The bar() function is empty. The basic idea is to implement and reverse the output of foo() in the bar() function and feed it the second encoded line. Please see the nut.js for details.
